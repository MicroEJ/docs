Usage
=====

To use the MWT library, add `MWT library module`_ to the project build file:

.. tabs::

   .. tab:: Gradle (build.gradle.kts)

      .. code-block:: java

         implementation("ej.library.ui:mwt:3.3.0")

   .. tab:: MMM (module.ivy)

      .. code-block:: xml

         <dependency org="ej.library.ui" name="mwt" rev="3.3.0"/>

.. _MWT library module: https://repository.microej.com/modules/ej/library/ui/mwt/
