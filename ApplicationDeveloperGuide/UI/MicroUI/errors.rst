Error Messages
==============

When an exception is thrown by the implementation of the MicroUI API, the exception `MicroUIException`_  with the error message ``MicroUI:E=<messageId>`` is issued, where the meaning of ``<messageId>`` is defined in the *Field Detail* of each error code.

.. _MicroUIException: https://repository.microej.com/javadoc/microej_5.x/apis/ej/microui/MicroUIException.html
