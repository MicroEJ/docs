Get Started with Graphical User Interface 
=========================================

This tutorial explains how to get started with GUI development in MICROEJ SDK. 
It goes through the key concepts needed by an application developer when building the GUI of a product.

In this tutorial, the target device is the `STM32F7508-DK <https://www.st.com/en/evaluation-tools/stm32f7508-dk.html>`_, 
a Discovery Kit by ST featuring a 4.3-inch 480x272 color LCD-TFT with a capacitive touch screen.

No board is required to follow this tutorial. Everything
can be tested in the :ref:`Virtual Device <section.virtual.device>`.

The tutorial should take XXX hours to complete (excluding the
installation time of MicroEJ SDK).


Intended Audience
-----------------

The audience for this document is Application engineers who want to create GUI applications for embedded devices.

The following document assumes that the reader is familiar with Java development.

Prerequisites
-------------

* MicroEJ SDK version 5.3.0 or higher. Check the :ref:`SDK User Guide <sdk_install>` for download and installation instructions.
