=====================================
VEE Port Creation for a Custom Device
=====================================

Description
===========

In this training, you will follow the typical steps followed by
a Firmware developer integrating MICROEJ VEE on 
a target device.

Intended Audience
=================

This training is for developers wishing to familiarize themselves with the MicroEJ Development flow.

Prerequisites
=============

- Good knowledge of :ref:`training_veeport_and_application_development`.
- A computer that can run MICROEJ SDK (see :ref:`system-requirements`).

Training Resources
==================

This training is composed of 2 units:

- Unit 1: step by step training that explains how to create a VEE Port for a custom device.
- Unit 2: continuation of Unit 1, create VEE Port Build and Run scripts to ease VEE Port development and validation.
